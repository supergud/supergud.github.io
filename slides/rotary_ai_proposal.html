Why Now

提案緣起：為什麼現在需要 AI？

  • AI 已經從聊天工具，進入日常工作流程
  • 扶輪社有大量社務、活動、公益與溝通需求
  • 社員來自不同產業，應用場景非常豐富
  • 現在導入 AI，可以提升社團效率與影響力
⚙️→🤖
現在的工作方式 → AI 協作方式
Rotary Fit

扶輪社最適合 AI 的原因

固定流程多

例會、活動、公益專案、會員經營，都能被整理成流程。

產業背景多元

社員來自各產業，能延伸出跨產業 AI 應用案例。

重視服務與連結

AI 減少行政負擔，把時間留給人與人的互動。

社務與本業雙效益

同時提升社團效率與社員本業競爭力。

社務 × 產業 × 服務

社務效率提升

Method

企劃書、SOP、新聞稿、會議紀錄

Sheets

客戶名單、活動報名、案件追蹤、統計分析

Forms

報名、預約、需求調查、滿意度問卷

Slides

活動提案、成果報告、年度簡報

Gmail

客戶回覆、提醒信、感謝信

Calendar

例會、預約、回訪、交期提醒

Drive

文件歸檔、活動照片、專案資料

NotebookLM

知識庫、產品手冊、SOP 問答

使用大家已經熟悉的工具落地
Automation

n8n 自動化可以做到什麼？

Trigger 觸發→讀取資料→AI 產生內容→條件判斷→通知 / 更新→留存紀錄
社務通知

活動報名確認、例會前一天提醒、生日關懷。

內容產出

會議摘要、活動新聞稿、社群貼文。

資料整理

滿意度問卷整理、活動成果統計、追蹤紀錄。

把重複流程變成自動化
AI Agent

AI Agent / Codex / Claude Code 的角色

  • 產生 Google Apps Script
  • 製作簡單活動網頁
  • 產生 WordPress shortcode 或 CSS
  • 協助串接 API
  • 建立報名表小工具
  • 修正自動化流程中的程式問題
適合對象

進階學員、資訊主委、網站管理者，或想做內部工具的企業主。

Apps ScriptWordPressAPI小工具

Three Stages

三階段課程提案總覽

2 小時演講版

適合例會邀請，重點是觀念導入與案例展示。

3 小時工作坊版

適合半日課程，讓社員實際產出 Prompt、表單與流程圖。

6 小時進階實作版

適合幹部訓練或深度導入，重點是 Google、n8n 與 AI Agent 實作。

依照社團需求，彈性選擇課程深度
